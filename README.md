# ☕ Coffee Sales Analysis Dashboard

## 📖 Overview

The Coffee Sales Analysis Dashboard is an interactive Microsoft Excel project designed to analyze and visualize coffee sales data. This dashboard transforms raw sales records into meaningful business insights, helping users understand sales performance, customer behavior, and market trends.

The project demonstrates practical applications of data analysis, dashboard design, and business intelligence using Microsoft Excel.

---

## 🎯 Project Objectives

- Analyze coffee sales performance over time.
- Identify top-performing customers.
- Compare sales across different countries.
- Evaluate the performance of various coffee products.
- Create an interactive dashboard for business decision-making.
- Present insights through easy-to-understand visualizations.

---

## 📊 Dashboard Features

### Sales Trend Analysis
- Monthly sales performance tracking
- Year-over-year sales comparison
- Revenue growth visualization

### Customer Analysis
- Top 5 customers based on total sales
- Customer contribution to overall revenue

### Country-wise Performance
- Sales comparison across different countries
- Geographic performance analysis

### Product Analysis
- Coffee type sales distribution
- Performance of:
  - Arabica
  - Excelsa
  - Liberica
  - Robusta

### Interactive Filters
- Timeline filtering
- Product filtering
- Country filtering
- Dynamic dashboard interaction using slicers
